## Design System

### Colors
- **Background**: Deep zinc/black (#09090b, #0f0d1a)
- **Surface**: zinc-900 with subtle borders
- **Primary**: Violet-600 (#7c3aed)
- **Success**: Green for "Done" badges, "Viral Score"
- **Processing**: Violet/purple pulsing

### Typography
- **Font**: Inter (clean sans-serif)
- **Headline**: Bold, large white
- **Body**: Muted grey
- **Monospace**: For technical text

### Components
1. **Sidebar** - Collapsed icon navigation, violet highlight
2. **Stat Cards** - Big number, small label, thin borders
3. **Project Table** - Thumbnail, title, date, duration, status badge
4. **Viral Score Badge** - Green pill with score
5. **Timeline** - Audio waveform, purple/grey, cut markers
6. **Export Button** - Full-width violet gradient
7. **AI Processing** - Circular particle animation, progress bar
